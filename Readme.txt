====================================================================
使用する際は，
controller_and_xmlをサーバPCに，

service\SIGNIUserTracker\SIGNiUserTracker_src_2-1-0を
C:\Program Files (x86)\OpenNI\Samplesに移動して下さい．
（移動先は各自環境に合わせて変更して下さい．）
====================================================================




====================================================================
各サービスのSIGファイルは以下の箇所にあります．
SIGNIUserTracker
service\SIGNIUserTracker\Samples\Bin\Release

VoiceControll
service\VoiceControll\Debug

MemorizeStorageSpace
\service\MemorizeStorageSpace\Debug
====================================================================




====================================================================
MemorizeStorageSpaceで使用する対応表"StorageSpaceList.txt"は
\service\MemorizeStorageSpace\MemorizeStorageSpace内に作成されます．
====================================================================




====================================================================
シミュレーションの流れ
----------------------1.片付け開始------------------------
(話す言葉)
命令：『片づけます』


---------------2.捨てる・置く物を渡す---------------------
右手でテーブルの上の物を掴んでロボットの右手に渡す


---------------3.捨てる・置く場所を教える-----------------
（話す言葉）　　　　　　　　（対応する場所）
命令：『赤色のゴミ箱』　　　場所：燃えるゴミ
命令：『青色のゴミ箱』　　　場所：びん・かん
命令：『緑色のゴミ箱』　　　場所：リサイクル・ペットボトル
命令：『茶色のワゴン』　　　場所：ワゴン


---------------------4.片付け終了-------------------------
（話す言葉）
命令：『終わります』
